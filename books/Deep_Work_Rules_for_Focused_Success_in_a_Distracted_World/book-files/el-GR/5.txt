Επεισόδιο 4: Κανόνας 1 - Εργαστείτε βαθιά

Στο Μέρος 2 της βαθιάς δουλειάς του Cal Newport: Κανόνες για την εστιασμένη επιτυχία σε έναν αποσπασμένο κόσμο, βυθίζουμε τον Κανόνα 1: Εργαζόμαστε βαθιά.Αυτός ο κανόνας χρησιμεύει ως το θεμέλιο για την καλλιέργεια της ικανότητας να συμμετέχει σε βαθιά εργασία αποτελεσματικά.

Κατανόηση του κανόνα 1: Εργαστείτε βαθιά

Ο Cal Newport υποστηρίζει ότι η βαθιά δουλειά δεν είναι μια δεξιότητα που μπορείτε να αναπτύξετε εν μία νυκτί.Αντ 'αυτού, είναι μια πρακτική που απαιτεί πρόθεση, αφοσίωση και δομημένη προσέγγιση.Ο κανόνας 1, εργάζεται βαθιά, παρέχει καθοδήγηση για το πώς να βυθιστείτε σε βαθιά εργασία με συνέπεια.

Οι τέσσερις κλάδοι της βαθιάς εργασίας

Για να εργαστεί βαθιά, το Newport περιγράφει τέσσερις κλάδους:

Επιλέξτε τη φιλοσοφία βάθους σας: Το Newport εισάγει δύο φιλοσοφίες: μοναστικές και διμοδιαιές.Η μοναστική φιλοσοφία περιλαμβάνει την αφιερώνεται σε μακρές, αδιάλειπτες περιόδους σε βαθιά δουλειά, ενώ η φιλοσοφία των διχάλων περιλαμβάνει τη δημιουργία μικρότερων αλλά τακτικών περιόδων για βαθιά εργασία.Επιλέξτε τη φιλοσοφία που ευθυγραμμίζεται με τον τρόπο ζωής και τους στόχους σας.

Αγκαλιάστε το τελετουργικό: Η δημιουργία ενός τελετουργικού βαθιάς εργασίας βοηθά να σηματοδοτήσει το μυαλό σας ότι ήρθε η ώρα να εμπλακείτε βαθιά.Αυτό θα μπορούσε να περιλαμβάνει μια συγκεκριμένη τοποθεσία, μια καθορισμένη ώρα και μια ρουτίνα που σας προετοιμάζει για εστιασμένη εργασία.

Κάντε μεγάλες χειρονομίες: Δημιουργήστε ένα περιβάλλον που υποστηρίζει βαθιά δουλειά.Το Newport συζητά πώς ορισμένα επιτυχημένα άτομα κάνουν σημαντικές αλλαγές, όπως η απομόνωση σε μια καμπίνα για ένα μήνα, για να επιτρέψουν τη βαθιά δουλειά.Ενώ αυτό μπορεί να μην είναι εφικτό για όλους, ακόμη και μικρές αλλαγές στον χώρο εργασίας σας μπορούν να κάνουν τη διαφορά.

Μην εργάζεστε μόνοι: Συνεργατική βαθιά δουλειά, ενώ είναι λιγότερο συνηθισμένη, μπορεί να είναι ένα ισχυρό εργαλείο.Συνεργαστείτε με άλλους για να αντιμετωπίσετε σύνθετα προβλήματα, να αξιοποιήσετε τις δεξιότητες του άλλου και να επιτύχετε συλλογικά βαθιούς στόχους εργασίας.

Η ρυθμική φιλοσοφία

Σε αυτή την ενότητα, το Newport δίνει έμφαση στη ρυθμική φιλοσοφία, όπου προγραμματίζετε βαθιά εργασία σε σταθερούς χρόνους.Με το να κολλήσετε σε ένα καθορισμένο πρόγραμμα, κάνετε βαθιά δουλειά μια συνήθεια, επιτρέποντάς σας να εκτελέσετε τα καλύτερα σας κατά τη διάρκεια αυτών των περιόδων.

Η δημοσιογραφική φιλοσοφία

Η δημοσιογραφική φιλοσοφία είναι πιο ευέλικτη.Αντί για σταθερά χρονοδιαγράμματα, εκμεταλλευτείτε τις ευκαιρίες να εργαστείτε βαθιά όταν προκύπτουν.Ενώ αυτή η προσέγγιση μπορεί να λειτουργήσει για μερικούς, απαιτεί πειθαρχία για να εξασφαλιστεί ότι η βαθιά εργασία δεν παραμελείται.

Δημιουργία τελετουργίας

Το Newport υπογραμμίζει τη σημασία ενός τελετουργικού εργασίας πριν από τη βαθιά.Αυτή η ρουτίνα σηματοδοτεί τον εγκέφαλό σας ότι ήρθε η ώρα να μεταβεί σε βαθιά λειτουργία εργασίας.Μπορεί να περιλαμβάνει απλές δραστηριότητες όπως η απενεργοποίηση των ειδοποιήσεων, το κλείσιμο των εφαρμογών αποσπάσματος και ο καθορισμός σαφών στόχων για τη συνεδρία της βαθιάς εργασίας σας.

Τα οφέλη των τελετουργιών

Η καθιέρωση ενός τελετουργικού μπορεί να είναι ένας παίκτης αλλαγής παιχνιδιού.Ακολουθώντας σταθερά ένα σύνολο ενεργειών πριν από τη βαθιά εργασία, το μυαλό σας αναγνωρίζει το πρότυπο και γίνεται προετοιμασμένη για εστιασμένη εργασία υψηλής αξίας.
