Επεισόδιο 6: Διδάξτε αυτό που γνωρίζετε

Στο Κεφάλαιο 6: Διδάξτε αυτό που γνωρίζετε, ο Austin Kleon υπογραμμίζει τη δύναμη της ανταλλαγής γνώσεων και εμπειρογνωμοσύνης ως μέσου τόσο για τη μάθηση όσο και για την οικοδόμηση μιας κοινότητας γύρω από το δημιουργικό έργο σας.

Ο Kleon πιστεύει ότι η διδασκαλία αυτού που γνωρίζετε δεν είναι μόνο μια γενναιόδωρη πράξη, αλλά και ένας πολύτιμος τρόπος για να εδραιώσετε τη δική σας κατανόηση ενός θέματος.Όταν διδάσκετε τους άλλους, είστε αναγκασμένοι να διατυπώσετε σαφώς τις ιδέες σας, να απαντήσετε σε ερωτήσεις και να συνθέσετε τις γνώσεις σας.Αυτή η διαδικασία μπορεί να εμβαθύνει τη δική σας εμπειρία και να σας βοηθήσει να βελτιώσετε τη δημιουργική σας πρακτική.

Το κεφάλαιο ενθαρρύνει τους δημιουργούς να εξετάσουν διάφορους τρόπους για να μοιραστούν τις γνώσεις τους, όπως τα γραπτά μαθήματα, τη δημιουργία βίντεο για τα βίντεο, τα κορυφαία εργαστήρια ή απλά να μοιραστούν πληροφορίες για τα ιστολόγιά τους ή τις πλατφόρμες κοινωνικών μέσων.Με αυτόν τον τρόπο, γίνεστε ένας πολύτιμος πόρος για άλλους που ενδιαφέρονται για τον τομέα της εμπειρογνωμοσύνης σας.

Ο Kleon υπογραμμίζει επίσης τη σημασία της αυθεντικότητας στη διδασκαλία.Δεν χρειάζεται να προσποιείτε ότι έχετε όλες τις απαντήσεις.Είναι εντάξει να είσαι εκπαιδευτής-δάσκαλος που μοιράζεται αυτό που έμαθα μέχρι στιγμής.Αυτή η προσέγγιση μπορεί να κάνει τη διδασκαλία σας πιο αναλογική και προσιτή σε άλλους.
