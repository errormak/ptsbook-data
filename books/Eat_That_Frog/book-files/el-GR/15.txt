Επεισόδιο 14: Βάλτε την πίεση στον εαυτό σας

Στο βιβλίο τρώτε αυτό το βάτραχο!Από τον Brian Tracy, η έννοια της πίεσης στον εαυτό σας είναι να χρησιμοποιήσετε το θετικό άγχος και τα κίνητρα για να επιτύχετε τους σημαντικότερους στόχους σας.Εδώ είναι πώς να εφαρμόσετε αυτή την έννοια αποτελεσματικά:

1. Αγκαλιάστε το θετικό άγχος:

Καταλάβετε ότι δεν είναι όλο το άγχος.Το θετικό άγχος, που συχνά αναφέρεται ως eustress, μπορεί να είναι ένα ισχυρό κίνητρο.Μπορεί να σας ωθήσει για να επιτύχετε περισσότερα και να προσπαθήσετε για αριστεία.
2. Ορίστε φιλόδοξους στόχους:

Στόχος υψηλό και θέτοντας προκλητικό, αλλά εφικτό, στόχοι για τον εαυτό σας.Έχοντας σαφείς στόχους θα δημιουργήσει μια αίσθηση επείγουσας ανάγκης.
3. Δημιουργήστε προθεσμίες:

Αναθέστε τις προθεσμίες στους στόχους και τις εργασίες σας.Αυτό προσθέτει μια αίσθηση επείγουσας ανάγκης και σας βοηθά να δώσετε προτεραιότητα στην εργασία σας.
4. Δημιουργία λογοδοσίας:

Μοιραστείτε τους στόχους σας με κάποιον που μπορεί να σας κρατήσει υπεύθυνο, είτε πρόκειται για μέντορα, φίλο ή συνάδελφο.Γνωρίζοντας ότι άλλοι γνωρίζουν τους στόχους σας μπορούν να σας παρακινήσουν να παραμείνετε σε καλό δρόμο.
5. Οπτικοποιήστε την επιτυχία:

Φανταστείτε τον εαυτό σας με επιτυχία ολοκληρώνοντας τα καθήκοντά σας και επιτύχετε τους στόχους σας.Η απεικόνιση μπορεί να ενισχύσει την εμπιστοσύνη και την οδήγηση.
6. Καταρρίψτε τους στόχους:

Διαχωρίστε τους μεγαλύτερους στόχους σας σε μικρότερες, διαχειρίσιμες εργασίες.Αυτό τους καθιστά λιγότερο συντριπτικό και σας επιτρέπει να εστιάσετε σε ένα βήμα κάθε φορά.
7. Εστίαση σε βασικά καθήκοντα:

Προσδιορίστε τα καθήκοντα που θα έχουν τον σημαντικότερο αντίκτυπο στους στόχους σας.Συγκεντρώστε τις προσπάθειές σας σε αυτά τα αντικείμενα υψηλής προτεραιότητας.
8. Εξαλείψτε τις περισπασμούς:

Ελαχιστοποιήστε τις περισπασμούς δημιουργώντας ένα εστιασμένο περιβάλλον εργασίας.Απενεργοποιήστε τις ειδοποιήσεις, κλείστε περιττές καρτέλες και δώστε προτεραιότητα στη βαθιά εργασία.
9. Χρησιμοποιήστε θετικές επιβεβαιώσεις:

Αναπτύξτε τη θετική αυτο-ομιλία και τις επιβεβαιώσεις που ενισχύουν την ικανότητά σας να πετύχετε.Η επανάληψη αυτών των επιβεβαιώσεων μπορεί να ενισχύσει την εμπιστοσύνη σας.
10. Ανταμοιβίστε τον εαυτό σας:
Γιορτάστε τα επιτεύγματά σας, ανεξάρτητα από το πόσο μικρά.Οι ανταμοιβές μπορούν να χρησιμεύσουν ως θετική ενίσχυση και να σας κρατήσουν κίνητρα.
11. Μείνετε πειθαρχημένοι:
Αναπτύξτε ισχυρή αυτοπειθαρχία και αυτοέλεγχο.Αυτό θα σας βοηθήσει να παραμείνετε αφοσιωμένοι στους στόχους σας, ακόμη και όταν αντιμετωπίζετε προκλήσεις.
12. Μάθετε από τις αποτυχίες:
Εάν αντιμετωπίσετε αποτυχίες ή αποτυχίες, θεωρήστε τις ευκαιρίες για μάθηση και ανάπτυξη.Ρυθμίστε την προσέγγισή σας και συνεχίστε να προχωράτε.
13. Μείνετε συνεπείς:
Η συνέπεια είναι το κλειδί για τη διατήρηση της πίεσης στον εαυτό σας.Κολλήστε τις ρουτίνες σας και συνεχίστε να σημειώνετε πρόοδο, μέρα με τη μέρα.
14. Λάβετε υπόψη τη μεγάλη εικόνα:
Θυμηθείτε γιατί ορίζετε αυτούς τους στόχους στην πρώτη θέση.Κρατώντας τη μεγάλη εικόνα στο μυαλό σας θα σας βοηθήσει να παραμείνετε κίνητρα σε δύσκολες στιγμές.
