에피소드 18 : 슬라이스와 작업을 주사위합니다

그 개구리를 먹으십시오!브라이언 트레이시 (Brian Tracy)에 의해, 슬라이스와 주사위의 개념은이 과제는 복잡한 작업을 작고 관리하기 쉬운 구성 요소로 분류하여 생산성을 높이고 압도적 인 느낌을 줄이는 것을 강조합니다.작업을 효과적으로 슬라이스하고 주사위하는 방법은 다음과 같습니다.

1. 작업 분석 :

완료해야 할 작업을 분석하여 시작하십시오.그 범위, 요구 사항 및 종속성을 이해하십시오.
2. 하위 작업 식별 :

작업을 더 작은 하위 작업 또는 단계로 나눕니다.이는 작업의 전반적인 완료에 기여하는 뚜렷한 조치가되어야합니다.
3. 하위 작업 우선 순위 :

각 하위 작업의 우선 순위를 결정하십시오.일부는 다른 것보다 더 중요하거나 시간에 민감 할 수 있습니다.
4. 작업 목록 작성 :

모든 하위 작업을 논리적 순서로 포함하는 작업 목록을 작성하십시오.이 목록은 더 큰 작업을 완료하기위한 로드맵 역할을합니다.
5. 이정표 설정 :

작업 내에서 주요 이정표를 식별하십시오.이정표는 전반적인 발전을 추적하는 데 도움이되는 중요한 진보 지점입니다.
6. 시간 할당 :

각 하위 작업에 필요한 시간을 추정하십시오.시간 할당에서 현실적입니다.
7. 하나의 하위 작업에 중점을 둡니다.

한 번에 하나의 하위 작업을 완료하는 데 집중하십시오.효율성을 줄일 수 있으므로 멀티 태스킹을 피하십시오.
8. 산만 제거 :

방해 요소를 최소화하고 각 하위 작업에 대해 집중된 작업 환경을 만듭니다.알림을 끄고 현재 단계에 최선을 다하십시오.
9. 진행 상황 :

하위 작업을 완료하면 작업 목록에서 표시하십시오.추적 진행 상황은 성취감을 제공합니다.
10. 업적 축하 :
하위 작업을 완료하거나 이정표에 도달하면 작은 승리를 축하하십시오.긍정적 인 강화는 동기 부여를 향상시킵니다.
11. 재평가 및 조정 :
작업 목록과 진행 상황을 정기적으로 검토하십시오.변경 또는 예기치 않은 문제를 수용하기 위해 필요한 경우 계획을 조정하십시오.
12. 필요한 경우 지원을 구하십시오.
전체 프로세스를 신속하게 할 수있는 경우 도움을 받거나 다른 하위 작업을 다른 사람에게 위임하는 것을 망설이지 마십시오.
