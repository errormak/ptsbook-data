Επεισόδιο 4: Εξάσκηση παραγωγικότητας χωρίς άγχος

Σε αυτό το επεισόδιο, εξερευνούμε το τμήμα του David Allen να κάνει τα πράγματα με τίτλο που ασκεί την παραγωγικότητα χωρίς άγχος.Αυτό το μέρος του βιβλίου ανακαλύπτει τις βασικές αρχές και τις μεθόδους για τη διαχείριση των καθηκόντων και των δεσμεύσεών σας με ελάχιστο άγχος.

Ο David Allen τονίζει αρκετές βασικές έννοιες σε αυτό το μέρος του βιβλίου:

Συμπληρώστε τα πάντα: Μία από τις θεμελιώδεις αρχές της παραγωγικότητας χωρίς άγχος είναι να συλλάβετε όλες τις δεσμεύσεις, τα καθήκοντά σας και τις ιδέες σας σε ένα εξωτερικό σύστημα.Αυτό θα μπορούσε να είναι ένα φυσικό σημειωματάριο, μια ψηφιακή εφαρμογή ή οποιοδήποτε εργαλείο που λειτουργεί για εσάς.Κάνοντας αυτό, απελευθερώνετε τον ψυχικό χώρο και εξασφαλίζετε ότι τίποτα σημαντικό να γλιστράει μέσα από τις ρωγμές.

Διευκρινίστε τις επόμενες ενέργειες: Αντί για ασαφή στοιχεία λίστας εργασιών, διευκρινίστε τις επόμενες φυσικές ενέργειες που απαιτούνται για την προώθηση των εργασιών προς τα εμπρός.Αυτό το επίπεδο κοκκώδους διευκολύνει την αντιμετώπιση των καθηκόντων χωρίς ασάφεια.

Οργανώστε τις εργασίες ανά πλαίσιο: ομαδοποιήστε τα καθήκοντά σας από το πλαίσιο στο οποίο μπορείτε να τις ολοκληρώσετε.Για παράδειγμα, οι εργασίες που απαιτούν μια τηλεφωνική κλήση θα πρέπει να ομαδοποιούνται, ώστε να μπορείτε να κάνετε αποτελεσματικά όλες τις κλήσεις σας ταυτόχρονα.

Αναθεωρήστε τακτικά: Η συνεπής αναθεώρηση των δεσμεύσεών σας και των εργασιών σας είναι ζωτικής σημασίας.Ο Allen συνιστά εβδομαδιαίες κριτικές για να διασφαλιστεί ότι το σύστημά σας παραμένει ενημερωμένο και ευθυγραμμισμένο με τις τρέχουσες προτεραιότητές σας.

Κάντε, μεταβιβάζετε, αναβάλλετε ή διαγράψτε: Όταν συναντάτε μια εργασία, αποφασίστε τι να κάνετε με αυτό.Μπορείτε να το κάνετε αμέσως, να το μεταβιβάσετε σε κάποιον άλλο, να το αναβάλλετε σε μεταγενέστερη ημερομηνία ή να το διαγράψετε εάν δεν είναι πλέον σχετικό.

Χρησιμοποιήστε ένα αξιόπιστο σύστημα: Το εξωτερικό σας σύστημα οργάνωσης, είτε πρόκειται για ψηφιακή εφαρμογή είτε για φυσικό σχεδιαστή, θα πρέπει να είναι αξιόπιστο και εύκολα προσβάσιμο.Θα πρέπει να εμπνεύσει την εμπιστοσύνη ότι όλα όσα πρέπει να θυμάστε έχουν συλληφθεί και διαχειρίζεται.
