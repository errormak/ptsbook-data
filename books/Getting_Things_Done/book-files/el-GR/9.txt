Επεισόδιο 8: Οργάνωση: Ρύθμιση των σωστών κουβάδων

Για να γίνουν τα πράγματα από τον David Allen, το Μέρος 2 συνεχίζει με την έννοια της οργάνωσης, εστιάζοντας ειδικά στη δημιουργία των σωστών οργανωτικών εργαλείων ή κουβάδων για να διαχειριστεί αποτελεσματικά τα καθήκοντά σας και τις δεσμεύσεις σας.Εδώ πρέπει να γνωρίζετε από αυτήν την ενότητα:

Η ανάγκη για οργανωτικά εργαλεία: Η οργάνωση αφορά τη δημιουργία μιας δομής για τη διαχείριση των καθηκόντων και των δεσμεύσεών σας αποτελεσματικά.Ο David Allen υπογραμμίζει ότι χρειάζεστε τα κατάλληλα εργαλεία, τόσο φυσικά όσο και ψηφιακά, για να εξασφαλίσετε ότι τίποτα δεν γλιστράει μέσα από τις ρωγμές.

Οι λίστες κλειδιών και οι κάδοι: Υπάρχουν αρκετοί λίστες ή κάδοι που σας βοηθούν να οργανώσετε την εργασία σας:

Επόμενες λίστες δράσεων: Πρόκειται για λίστες εργασιών που μπορείτε να ολοκληρώσετε αμέσως.Καταβάλλονται με βάση το πλαίσιο (π.χ. γραφείο, σπίτι, τηλέφωνο) για να διευκολύνετε να δείτε τι μπορείτε να κάνετε σε μια συγκεκριμένη κατάσταση.
Λίστες έργων: Κάθε έργο που έχετε εντοπίσει κατά τη διάρκεια της επεξεργασίας θα πρέπει να έχει μια αντίστοιχη λίστα έργων.Αυτή η λίστα παρακολουθεί όλες τις επόμενες ενέργειες που απαιτούνται για την προώθηση ενός έργου.
Ημερολόγιο: Χρησιμοποιήστε το ημερολόγιό σας για να προγραμματίσετε ραντεβού, συναντήσεις και οποιεσδήποτε ευαίσθητες στο χρόνο ενέργειες.Το ημερολόγιό σας πρέπει να περιέχει μόνο στοιχεία με συγκεκριμένες ημερομηνίες και ώρες.
Αναμονή για τη λίστα: Αυτή η λίστα σας βοηθά να παρακολουθείτε εργασίες ή δεσμεύσεις που περιμένετε να ολοκληρώσουν οι άλλοι.Εξασφαλίζει ότι παρακολουθείτε αυτά τα στοιχεία όπως απαιτείται.
Κάποια μέρα/ίσως λίστα: ιδέες, έργα ή δεσμεύσεις που δεν αποτελούν προτεραιότητα αυτή τη στιγμή, αλλά μπορεί να είναι στο μέλλον θα πρέπει να καταγράφονται σε αυτόν τον κατάλογο.
Σύστημα αρχειοθέτησης αναφοράς: Δημιουργήστε ένα οργανωμένο σύστημα αναφοράς για την αποθήκευση πληροφοριών και υλικών που ίσως χρειαστεί να αναφέρετε αργότερα.
Επιλέγοντας τα σωστά εργαλεία: Τα εργαλεία που επιλέγετε για την οργάνωση μπορεί να ποικίλουν από φυσικά φορητούς υπολογιστές και φακέλους σε ψηφιακές εφαρμογές και λογισμικό.Αυτό που είναι απαραίτητο είναι ότι ταιριάζουν στις προσωπικές σας προτιμήσεις και τον τρόπο ζωής σας.Ο David Allen προτείνει ότι τα εργαλεία σας πρέπει να είναι εύκολο να έχουν πρόσβαση, να είναι αξιόπιστα και να προσαρμοστούν στις ανάγκες σας.

Η εβδομαδιαία ανασκόπηση: Η διατήρηση του οργανωτικού σας συστήματος απαιτεί τακτική ανασκόπηση.Η εβδομαδιαία αναθεώρηση είναι μια κρίσιμη πρακτική GTD.Κατά τη διάρκεια αυτής της περιόδου, διασφαλίζετε ότι όλες οι λίστες σας είναι ενημερωμένες, επαναξιολογήστε τις προτεραιότητές σας και προσαρμόστε τις δεσμεύσεις και τα έργα σας ανάλογα με τις ανάγκες.
